# 数据审查

> 文档维护：GPT-5（最近更新 2026-06-03）

本目录用于存放 ODS / GCS 数据审查提示词、执行约定和审查报告。

## 目录结构

- `ODS_GCS_DATA_AUDIT_PROMPT.md`：交给审查 Agent 使用的完整提示词。
- `reports/`：审查报告输出目录。每次审查应单独生成一份报告，记录审查时间、数据范围、审查 LLM、BigQuery / Tushare / Tinyshare 口径、样本范围、发现问题和限制。

## 本次审查范围

- 只审查 2019-01-01 及之后的数据。
- 审查动作只读：不得补数据、不得改写 GCS Parquet、不得重建 BigQuery ODS/DWD/DWS/ADS 表。
- 审查 Agent 可以自行编写和修改审查脚本、限速器、重试逻辑、抽样逻辑和报告生成代码；如果审查过程中发现请求参数、API 调用、并发、限速、schema 检测或报告代码有问题，应自行修正审查代码后继续审查。
