# 分析：long-only 现金/降仓 overlay 对 Sharpe 的真实贡献（NAV 级只读）

## 问题

确认"适当空仓"（long-only 兼容的现金 / 降仓 overlay）对策略 Sharpe 的真实贡献——
它是 **Sharpe 杠杆**，还是只是 **稳健 / 回撤（Calmar）层**？

## 数据与方法

- Baseline backtest：`bt_s1_annual_roll_continuous_true5y_2021_2026_n20_w075_v20260611_01_ca01`（true-five-year CA-on 研究 baseline，**≠ accepted**）。
- 长窗：`2021-01-04` 至 `2026-06-09`；近窗：`2024-01-02` 至 `2026-04-30`。
- 收益源：`research_backtest_nav_daily.daily_return`（已含策略自身交易成本）。
- overlay 公式：`r'(t) = e(t) * r(t) - cost(t)`，现金部分收益 = 0，`e(t) ∈ [0, 1]`（不加杠杆）。
- **严格无前视**：所有 exposure 信号都用 lagged（截至 `t-1`）数据。vol-targeting 用
  `exposure = clip(min(1, target_ann_vol / trailing_vol_{t-1}), 0, 1)`，trailing vol 为
  20/60 日 lagged 年化样本标准差；trailing-drawdown 用 lagged 20 日累计收益；market-state
  用 `dws_market_state_daily`（`market_state_v1_20260607`）的 `is_smallcap_trend_down` /
  `csi1000_drawdown_20d`，均 shift(1)。warm-up（无 vol 估计）期 exposure=1，不注入伪 skill。
- exposure 路径在**完整长窗**上构建（近窗指标用同一路径切片，避免冷启动），近窗 NAV 重置为 1.0。
- 两个 Sharpe 口径并报：**daily Sharpe** = `mean(r)/std(r)*sqrt(252)`（长窗 baseline 锚点 `0.737`）；
  **contract Sharpe** = `CAGR / annual_vol`（长窗 baseline 锚点 `0.6685`，近窗 `~1.46`）。
- overlay 自身换手成本：按 `abs(delta exposure) * cost_bps` 粗扣（grid `0,10,20` bps），
  零成本先做再看净 Sharpe。10/20 bps 对应项目成本口径（佣金~1bp + 卖出印花 5bps + 滑点 5bps/边）下
  单位 |delta exposure| 的换手成本量级。

## 锚点与恒等校验

- baseline（exposure≡1、零成本）恒等复现：长窗 daily Sharpe `0.736915`、
  contract Sharpe `0.668208`、MaxDD `-37.43%`、
  Calmar `0.4101`；近窗 daily Sharpe `1.337033`、
  contract Sharpe `1.4168`、Calmar `1.0688`。
- 与任务给定锚点一致（长窗 `0.737` / `0.6685`，近窗 daily `1.337` / contract `~1.46`）。

## 局限性

NAV 级线性缩放上界估计，**不是真实可交易回测**：忽略 lot 取整、成分股卖出可达性、现金再分配与
选股的交互、冲击成本，以及 exposure 改变对后续持仓结构的反馈。grid 内"最优"变体在同一段历史上
选出，存在 in-sample selection bias，会高估样本外上界。结论只用于回答"机制天花板"的定性问题，
不作为可实现收益、不改默认 profile、不 promotion、不标 accepted。

## 关键读数（零成本）

- baseline 长窗 daily Sharpe = `0.7369`，Calmar = `0.4101`，MaxDD = `-37.43%`。
- 所有 overlay 中**长窗 daily Sharpe 最高**：`ddderisk_lb20_thr-0.1_elow0_cost0bps` →
  Sharpe `0.7948`（较 baseline +0.0578），
  Calmar `0.4965`，MaxDD `-31.44%`。
- 所有 overlay 中**长窗 Calmar 最高**：`ddderisk_lb20_thr-0.05_elow0_cost0bps` →
  Calmar `0.5286`（较 baseline +0.1184），
  MaxDD `-26.88%`（较 baseline 改善 -10.55pp），
  Sharpe `0.7656`。

## 结论：适当空仓是「稳健 / Calmar 层」，不是 Sharpe 杠杆

**核心判定：long-only 现金 / 降仓 overlay 对 Sharpe 的贡献很小且脆弱，主要价值在压回撤（MaxDD / Calmar）。它无法把长窗 `0.7369` 的 Sharpe 抬向 `1.2 / 1.5`。** 三个机制各自的证据：

1. **vol-targeting（教科书式 Sharpe 杠杆）在本策略上失效——它降 Sharpe 也降 Calmar。** 全部 8 个 vol-target 变体长窗 daily Sharpe 都低于 baseline（`0.4160 ~ 0.6714` vs `0.7369`），Calmar 也全部下降（`0.2054 ~ 0.3575` vs `0.4101`）。原因：本小盘 alpha 的「高波动期」并不是「坏收益期」，按波动率降仓等于在仍有上行的高波动段砍掉暴露，既丢收益又没换到回撤改善。**结论：波动率不是本策略坏收益的前瞻预测量，vol-targeting 不是这里的 Sharpe 工具。**

2. **唯一能把 Sharpe 推高的是「回撤动量」降仓（跌过阈值才降仓），但幅度小、且对成本敏感。** 最优 `ddderisk_lb20_thr-0.1_elow0_cost0bps`：零成本长窗 daily Sharpe `0.7948`（较 baseline **仅 +0.0578**）、contract Sharpe `0.7418`（+0.0736，零成本下刚够到 `0.70` 门）、MaxDD `-31.44%`、Calmar `0.4965`。该信号慢（仅 21 次切换），20bps 成本后仍保住 Sharpe `0.7564` / contract `0.6977`——**净增益收窄到约 +0.02 daily / +0.03 contract**。更紧的 `-5%` 阈值切换多（51 次），20bps 后 Sharpe 反跌到 `0.6666`（低于 baseline）。即便取这个 in-sample 最优上界，Sharpe 也只在 `0.7~0.8` 区间，离 `1.2 / 1.5` 差一个数量级。

3. **market-state（外生信号）降仓基本是纯回撤 / 稳健层。** `smallcapdown_elow0_cost0bps` 把 MaxDD 从 `-37.43%` 压到 `-20.38%`（全场最佳回撤改善）、Calmar 升到 `0.5225`，但 daily Sharpe 反而降到 `0.6118`（且 65 次切换，20bps 后 Sharpe `0.4851`、Calmar `0.3437`，增益被成本吃掉）。`csi1000_drawdown_20d` 触发的版本表现略好（最优 daily Sharpe `0.7543`、Calmar `0.5015`），但同样是「换回撤、不换 Sharpe」。

4. **近窗的高 Sharpe（`~1.34 / 1.46`）不是空仓带来的，而是满仓策略本身在 2024–2026 子段的属性。** baseline（零 overlay、全程满仓）近窗 daily Sharpe 已经是 `1.3370`、contract `1.4168`——没有任何降仓动作。所以「Sharpe 从 0.7369 升到 1.2/1.5」靠的是**子周期 alpha / regime**，不是现金 overlay。

**回答 owner 的问题：适当空仓主要是稳健层（压 MaxDD / 抬 Calmar）。** MaxDD 可从 `-37.43%` 压到最好 `-20.38%`（market-state）/ `-31.44%`（回撤动量），Calmar 从 `0.4101` 抬到最高 `0.5286`（零成本，含成本后回落到约 0.43~0.47）；但 Sharpe 最多只能买到约 `+0.06`（in-sample、零成本上界），实打实成本后约 `+0.02 ~ 0.03`，不构成 Sharpe 杠杆。这与既有结论一致：指数对冲对 Sharpe 无用（PR #179），exposure 上界最优 contract Sharpe 仅 `~0.748`（PRD_20260613_02）。**抬 Sharpe 的杠杆在 alpha 质量 / 信号 / 组合构造，不在暴露管理。**

### 口径与边界（须与结论并列声明）

1. **结论是条件性的，不是普适定律。** 成立范围限定为「long-only、`exposure ∈ [0,1]`、零成本、此类粗粒度 de-risk overlay（单阈值 / 单波动目标 / 单状态标志）」。两块未测空白区可能改变 Sharpe 结论：(a) `exposure > 1` 加杠杆放大好 regime（任务明确排除）；(b) 比这四个更聪明的外生择时信号。四个 overlay 失败不等于「不存在任何能提 Sharpe 的择时信号」。
2. **「小且脆弱」是定性判断，未做正式显著性检验。** overlay 与 baseline 的正确检验是 paired / 相关-Sharpe 差异检验（Jobson-Korkie / Ledoit-Wolf），不是边际 Sharpe SE；因最优 `ddderisk` 平均 exposure 高、切换少，与 baseline 路径高度重合，配对检验 SE 远小于边际 SE。本结论靠「delta 算术 + 成本敏感性」而非 t 值——对「能否抬到 1.2/1.5」这一量级问题足够，但不宣称统计显著性。
3. **回撤降仓的 Sharpe 增益是 in-sample best-of-grid 上界。** lookback / 阈值 / e_low 均可调，无样本外 / 交叉验证 / 多重检验校正；20bps 成本即崩到 `+0.02`、contract Sharpe 跌破 `0.70` 门，与「过拟合边际刚过噪声」一致——这反而**强化**「不是 Sharpe 杠杆」。

> 本结论已三路交叉验证（**PASS，无阻断**）：① 独立重算——另写 query+脚本（未导入本 harness、只读 SELECT）复现 4 个 headline 变体共 22 个指标，max abs diff `6.78e-6`；② 无前视审计——所有信号严格 `t-1` 滞后、`shift(1)` 位置正确、近窗切片因果；③ 方法学批判——核心结论在上述口径下成立，caveat 即本节三条。
