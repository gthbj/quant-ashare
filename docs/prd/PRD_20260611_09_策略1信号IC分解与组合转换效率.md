# PRD：策略 1 信号 IC 分解与组合转换效率分析

> 状态：草案，待 owner review。
> 范围声明：纯只读研究分析（BigQuery 只读 + 本地 pandas），不重训模型、不改组合构造、不改 ledger、不写任何 BigQuery dataset、不 promotion。
> 关联：`DECISION-20260611-02`（effective-window 研究口径）、PR #179（尾部风险 Overlay A/B）、PR #181（暴露管理上限仿真）、OQ-010。

---

## 1. 背景

风控/包装层的三轮实验已经收口：P1/P2 Overlay A/B（PR #179）与暴露管理无摩擦上限仿真
（PR #181，best Calmar `0.4077`、best contract Sharpe `0.6006`）共同证明，在当前 prediction
流上继续做止损/仓位类 overlay 没有增量空间。

与此同时，两层证据出现了方向相反的读数：

- **组合层**：official continuous 超额 IR `0.542`，t ≈ 0.542 × √5.43 ≈ **1.26**——组合 NAV
  层面的超额收益在统计上无法与运气区分；
- **信号层**：六个年度 refit 模型的 valid 日度 rank IC 全部为正（`0.039 ~ 0.098`，日度
  ICIR `0.30 ~ 0.81`，每年 ~237 个交易日；registry `oriented_valid_rank_ic_*` 字段），
  即使对 11 选 1 的选择偏差和 5 日重叠标签的自相关打折，跨六个独立年份同号的信号证据
  仍然显著强于组合层表现。

"信号显著真实 + 组合超额平庸"指向同一个假设：**alpha 在信号→组合的转换环节被浪费**
（long-only 丢空头侧、Top20 截断丢排序尾部、等权丢分数强度、满仓绑死 beta）。另有一个
独立疑点：2024-01-01~02-07 踩踏段超额 `-19.3pp`，怀疑信号隐性做多小市值拥挤风格。

本 PRD 把度量衡从组合 NAV 切换到信号层，回答两个问题：

1. **IC 分解**：信号的预测力住在哪（风格中性后还剩多少 / 哪种市场体制下失效 /
   分数分布的哪一段 / 多长的 horizon）？
2. **组合转换效率**：从"信号理论上限"到"当前官方组合"，每一条包装约束各损耗了多少 IR？

## 2. 目标

1. 在 official synthetic prediction 流上产出**真 OOS** 的 IC 分解报告（按年 / 风格中性 /
   市场体制 / 分数分位 / horizon 五个维度）。
2. 产出"转换阶梯"：L0 多空 → L1 宽 long-only → L2 窄分数加权 → L3 现行口径，逐级 IR
   落差表 + 直接测量的 TC（实际主动权重 vs 分数隐含权重的相关性）时间序列。
3. 按 §6 预登记解读规则给出结论，支撑 owner 的三个后续决策：① 组合构造是否改
   （加权方案 / 持仓数 / 风格约束）；② 是否立对冲结构可行性评估；③ 是否回炉模型
   （中性化后信号不足时）。

## 3. 非目标

- 不改 official 组合构造、不重跑 selection / refit / synthetic、不实现新 ledger 语义。
- 不重训或新增模型；中性化只作用于分析，不改训练标签（那是后续决策）。
- 不做实盘/模拟盘改动；不动 `dws_market_state_daily` 与任何生产口径。
- 阶梯组合是 NAV 级 paper portfolio（同 PR #181 方法），不是可交易回测；L0 多空仅作
  信号容量度量，**不**评估其可实施性（小资金无法做空个股）。

## 5. 实验设计

### 5.1 Part A：IC 分解

基础定义：日度横截面 Spearman rank IC（score vs fwd 5d return，universe 内，按
`score_orientation` 取向），窗口 `2021-01-04 ~ 2026-06-09`。同时输出 biweekly 调仓日
子样本的 IC（组合实际消费的频率）。

| 维度 | 切法 | 回答的问题 |
|---|---|---|
| A1 按年 | 6 个预测年分组 | 真 OOS IC 逐年稳定性（区别于 registry 的 valid 口径） |
| A2 风格中性 | score 与 fwd return 各自对 log(circ_mv)（+行业 dummy，如可用）做日度横截面 OLS 取残差，算残差 IC | 信号是真选股还是风格 beta 伪装 |
| A3 按体制 | 按 t-1 收盘 `market_regime` 三组 | 信号在跌市是否失灵（解释 crunch -19.3pp） |
| A4 按分位 | top 5% / top decile 组内 IC；多头侧（top half）vs 空头侧（bottom half）IC 贡献拆分 | long-only 拿得到的那段信号还剩多少 |
| A5 按 horizon | fwd 1/5/10/20d 四条 IC 衰减曲线 | biweekly（~10d）调仓的后半程有没有信号 |

**统计纪律（硬要求）**：重叠前向窗口导致 IC 序列自相关，所有 t 值必须用
Newey-West（lag ≥ horizon）或 block bootstrap（block ≥ horizon）修正，并报告有效样本量；
禁止用朴素 i.i.d. 标准误。

### 5.2 Part B：转换阶梯与 TC

四级 paper portfolio，全部复用同一预测流、同一 biweekly 调仓日、PIT 同 P2 约定，
NAV 级仿真（方法与 `simulate_exposure_overlay_upper_bound.py` 同族：本地 pandas、
恒等校验先行）：

| 级 | 构造 | 度量的约束 |
|---|---|---|
| L0 | 分数加权多空：top decile 多 / bottom decile 空，dollar-neutral | 信号理论容量（仅度量用） |
| L1 | long-only 分数加权 Top50 | 截断宽度 |
| L2 | long-only 分数加权 Top20 | 集中度 |
| L3 | long-only **等权** Top20 + 单票 7.5% cap | 现行口径（≈ official） |

- 每级输出：CAGR、年化波动、IR vs `000852.SH`（L0 另报绝对 Sharpe）、MaxDD、逐年分解；
  相邻级差 = 对应约束的定价。
- 主结果零成本；附 20bps 单边成本档敏感性（结论必须在两档下同向）。
- **L3 恒等校验**：L3 的逐年收益与 official continuous 实际结果对照，相关性与量级
  必须接近（允许 lot 取整 / 现金残留造成的偏差，需在报告量化该偏差）；偏差过大说明
  paper portfolio 构造有错，先修再出结论。
- **TC 直接测量**：每个调仓日算 corr(L3 / official 实际主动权重, 分数隐含权重)，
  输出时间序列与均值；并对 official 的 `research_portfolio_target_daily` 权重做同样计算，
  交叉验证 paper L3 与真实组合的一致性。

## 6. 预登记解读规则（先写后跑，禁止按结果反调）

| 观测 | 解读 → 指向的决策 |
|---|---|
| A2：市值中性后 IC ≥ 原始 60% | 信号以真选股为主 → 组合构造改进优先（B 部分指路） |
| A2：市值中性后 IC < 原始 40% | 信号以小市值风格暴露为主 → 回炉模型（中性化标签/特征）或显式风格约束，crunch 脆弱性视为已解释 |
| A3：risk_off 组 IC 不显著或翻负 | 跌市信号失灵 → 体制条件化（P(bear) 特征 / 条件暴露）从"可选"升为"必要" |
| A4：空头侧 IC 贡献 > 60% | long-only 拿不到主要信号 → 对冲结构可行性评估立项（小资金约束一并评估） |
| B：L0 无摩擦 IR < 1.0 | 信号容量本身有限 → 回 OQ-010 加强信号，组合改进收益有限 |
| B：L3−L2 落差 ≥ 0.1 IR | 等权改分数加权是近乎免费的改进 → 立组合构造小 PRD |
| B：L1−L2 落差 ≥ 0.1 IR | 放宽到 Top30~50 值得做 → 同上合并评估 |

灰区（介于阈值之间）如实报告，交 owner。多条规则同时触发时全部列出，不做合并简化。

## 8. 风险与控制

| 风险 | 控制 |
|---|---|
| 行业快照非 PIT 污染中性化 | 市值中性为主判据；行业版单列并标注局限 |
| 重叠窗口虚增显著性 | §5.1 统计纪律为硬门 |
| 前向收益口径与训练标签不一致导致 IC 失真 | §4.2 强制复用 panel 口径 |
| paper portfolio 构造错误 | L3 恒等校验先行，不过不出结论 |
| L0 多空被误读为可实施策略 | §3 非目标显式声明 + 报告重复声明 |
| 预测流本身的选择偏差（selected 模型） | 报告声明：分析对象是"现役信号"，结论限于该流 |
