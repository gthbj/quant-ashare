# 分析：策略1暴露管理上限仿真（NAV 级离线估计）

## 预登记判据

以最优变体的**无摩擦** Calmar 为准：

- Calmar < 0.5 -> 结论“ledger 工程缓做/降优先级，剩余缺口在 alpha 质量”；
- Calmar >= 0.7 -> 结论“建议立暴露管理 PRD，做真实 ledger 实现验证摩擦后残余”；
- 0.5 ~ 0.7 -> 灰区，列出取舍交 owner。

参考锚点：v3 gates 为 contract Sharpe >= 0.70、Calmar > 1.0；baseline Calmar 0.2646；A2（skip-buys 实跑）Calmar 0.2587。

## 数据与方法

- Baseline backtest: `bt_s1_annual_roll_continuous_2021_2026_n20_w075_v20260610_02`
- 窗口：`2021-01-04` 至 `2026-06-09`
- 市场状态版本：`market_state_v1_20260607`
- 基准：`000852.SH`，`pct_chg / 100`
- 仿真公式：`r'(t) = e(t) * r(t) - cost(t)`，现金部分收益为 0。
- PIT：`t` 日市场状态只影响下一开市日；首日无前序状态，暴露为 1。
- 调仓日变体：按 `build_candidates.sql` 的 ISO 年/周最后开市日、`MOD(week_idx-1,2)=0`、anchor `2021-01-04` 重建。
- 成本：每次暴露变化扣 `abs(delta exposure) * cost_bps / 10000`，成本在暴露变化日进入日收益。

## 局限性

这是 NAV 级无摩擦/低摩擦上界估计，不是真实可交易回测。线性缩放假设忽略 lot 取整、成分股卖出路径、选股与现金再分配交互、冲击成本、卖出不可交易约束，以及 exposure 改变后对后续持仓结构的反馈。无摩擦结果只能表示“机制天花板”，不能直接视为可实现收益。

## 恒等与覆盖校验

- `e(t) == 1.0`、零成本恒等仿真通过，复现 baseline：
  - CAGR `0.12036528993503293`
  - MaxDD `-0.45481511936569563`
  - Calmar `0.26464663290635421`
  - contract Sharpe `0.5285475500566128`
- market state / NAV / benchmark 均按 SSE 开市日逐日对齐，`is_risk_off` 无 NULL。
- baseline crunch excess vs `000852.SH` = `-0.19329880132544719`，与 PR #179 对比表一致。
- 表中 `information_ratio` 与 `excess_return_vs_000852` 按本任务指定的 `dwd_index_eod.pct_chg / 100` 重算；不复用 summary 表 legacy benchmark_return。
- Markdown 表与完整 CSV 使用同一详细列集；CSV：`docs/analysis_strategy1_exposure_overlay_upper_bound_20260611_results.csv`

## 结论

最优无摩擦变体：`two_state_biweekly_elow0_cost0bps`。

- CAGR: `12.13%`
- MaxDD: `-29.75%`
- Calmar: `0.4077`
- contract Sharpe: `0.6006`
- 平均暴露: `0.8874`
- 暴露切换次数: `24`

按预登记判据：**ledger 工程缓做/降优先级，剩余缺口在 alpha 质量**

所有 exposure 变体的最高 contract Sharpe 为 `0.6006`，仍低于 v3 gate `0.70`。即使暴露管理达到本次无摩擦上界，v3 双门仍不可达，主要缺口仍在 alpha 质量或信号/组合构造。
