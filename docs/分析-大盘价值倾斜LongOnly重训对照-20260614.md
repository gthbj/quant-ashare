## 4. 结论与建议

- **在不变的沪深主板 long-only 约束内,把模型从小盘反转重训为大盘价值倾斜,长窗 risk-adjusted 仅微升(+0.08 Sharpe)、近窗变差,未能向 1.5 靠。** 这与大盘 alpha 存活探针自身的 caveat 吻合:纯多头大盘价值只是「最现实的第一步」,**冲 1.2 大概率仍需大盘融券空头腿**;1.5 long-only 是 stretch-of-a-stretch,本轮证伪。
