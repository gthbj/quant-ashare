# 分析：大盘股 alpha 存活探针 —— 抬 Sharpe 的下一步方向证据

> 文档维护：Claude Opus 4.8（2026-06-13）
> 范围：纯只读 paper 探针（BigQuery 只读 + 本地 pandas）。不跑 live、不训练、不写 BigQuery/GCS、不改 ledger/默认 profile、不改 `.agent/memory`。
> 关联：本会话 Sharpe 归因诊断（transfer ladder / IC 分解，`analyze_signal_ic_decomposition.py`）；`scripts/strategy1/analyze_largecap_alpha_survival.py`；`docs/analysis_largecap_alpha_survival_20260613.csv`。

## 1. 背景与问题

Sharpe 归因诊断结论：v1 长窗 contract Sharpe 卡在 `0.6685`，绑定约束是 **long-only 结构作用在中证1000 小盘上**——信号 ~53% 的 edge 在空头侧（纯多头采不到），且对冲 beta 对 Sharpe 无效（实测 0.737→0.736）。owner 目标是把 Sharpe 抬到 **1.2 以上**，并提出「换模型 / 换大盘股」是否有希望。

本探针回答一个决定性的前置问题：**alpha 在大盘股里还存活吗？是否「纯多头可采」？** 这决定换大盘股是不是抬 Sharpe 的可信路径。

## 2. 方法

同一面板（`dws_stock_feature_daily_v0`，`feature_version=strategy1_pv_v0_20260601`）上，主板（SSE_MAIN/SZSE_MAIN）、可交易、`2021-01-04..2026-06-09`，按日 `total_mv_cny` 排名分三组：**大盘 top300 / 中盘 301-800 / 小盘 801-1800**。

- **per-feature 有符号 rank IC**（5d）：每日横截面 Spearman 秩相关（特征 vs 原始 fwd_ret），按组求均值。
- **composite**：固定**经济先验符号**的等权 z 分（反转 −ret_5d、低风险 −vol_20d/−turnover_z、价值 +ep_ttm/+bp/+divyield；无 in-sample 拟合符号）。
- **多空上界**：**非重叠**每 h 个交易日调仓，组内按 composite 排序，多空十分位差 / 纯多头十分位−组均值，年化 Sharpe。

**口径用市值排名而非指数成分**（项目无现成 300/800 成分表，但面板含全部大盘股，茅台/平安在列），更连续稳健。

## 4. 判读（三条结论，paper directional——非 hard decision 口径）

1. **alpha 在大盘股存活**：大盘 composite IC 5d `0.049` / 20d `0.078`，是小盘的 ~75%（5d）到 ~86%（20d）——更低但没塌，20d 的 0.078 是个可用的 IC。
2. **alpha 成分随市值变化**：**反转 / 低换手 / 低波**（典型小盘无效率溢价）在小盘强 2-3 倍、到大盘大幅衰减；**价值（ep/bp/股息）在大小盘一样强**。→ 大盘策略应是**价值倾斜**，不是反转倾斜。
3. **关键方向：纯多头可采的 alpha 随市值升高**：纯多头超额 Sharpe 大盘 `0.59`(20d) / 中 `0.20` / 小 `0.09`，5d 更极端（大盘 `0.42` / 小 `-0.04`）。**小盘的边几乎全在空头侧（纯多头采不到，正是 v1 卡 0.67 的根因）；大盘有相当一块在多头侧、不做空也能拿。** L/S 十分位则相反（小盘 0.69 > 大盘 0.31），但小盘不可融券、大盘可融券。该方向在 row-wise skip-NA 口径、`label_valid` 过滤、`in_universe_default` 对照下均不变号（详见 §6 caveat）。

## 5. 对 1.2 目标与路径的含义

- **换大盘股有真实价值**——理由不是「波动低」，而是：① alpha 在大盘**可纯多头采**（小盘不行）；② 大盘**可融券**（能加空头腿）。
- **纯多头大盘价值模型**（当前授权内、不需做空）是最现实的第一步，有望显著抬 Sharpe；**冲 1.2 大概率仍需大盘融券空头腿**（靠重训模型把多空做强）。
- **换模型**：维度对（Sharpe 是主动驱动）但单独到 1.2 希望小；正确用法是**在大盘价值域重训**，而非在小盘上继续调。

## 6. 诚实 caveat（结论读相对、不读绝对）

- composite 是 **6 个等权原始特征的粗代理**——训练好的模型强得多（小盘：模型多空 Sharpe ~3.4 vs 本 composite 0.68）。故**绝对 Sharpe 是下界，跨组的相对/方向才是稳健信号**；要确认大盘绝对上界须真重训。
- in-sample、零成本、**未扣融券费（~8-10%/年）**——多空上界乐观。
- 市值排名分组 ≈ 指数域但非精确指数成分；先验符号对齐避免了 in-sample 偏差，但 mom_60_20 等先验模糊特征未入 composite（单列 IC 仅供参考）。
- **complete-case / 有效覆盖率（PR #221 review 补充）**：composite 要求 6 个特征全非空（任一 NULL 该行作废），实际 complete-case 覆盖率大/中/小 = `87.2%` / `78.5%` / `67.0%`，故 20d Sharpe 的有效平均名字数约 `261 / 393 / 669`（非 300/500/1000）。大盘估值特征覆盖更全本身是个混杂项。row-wise skip-NA（不要求 6 特征全有）口径同方向：20d 纯多头超额 Sharpe 大/中/小 ≈ `0.448 / 0.104 / -0.108`——绝对数变、方向不变。**主表数值按 complete-case 口径读。**
- **统计不确定性（PR #221 review 补充）**：20d 非重叠仅 ~65 个 period，本探针**未给 CI / bootstrap / 年度子样本稳定性**。方向在多组对照下一致，但应作为 **paper directional evidence**，绝对 Sharpe 与"大盘 > 小盘"的幅度不作 hard decision 的精确依据；真重训 + 不确定性区间后再定量。
